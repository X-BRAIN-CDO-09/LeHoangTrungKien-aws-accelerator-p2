# Lab Runbooks

Folder này chứa runbook dùng cho show-and-tell và Test 2.

## Runbook hiện có

- `pod-compromise.md`: xử lý khi nghi pod bị compromise.
- `unsigned-image-rejected.md`: xử lý khi admission reject image vì thiếu signature.

Khi chạy lab, bổ sung evidence thật vào từng runbook: namespace, pod, image digest, event, lệnh đã chạy và kết quả.
